The Analysis of Feed-Material Processes in Manufacturing System with Empirical Data

Simulation Optimization

Abstract

This study has created a simulation system with python to compare two different feed-material processes (named method A, and method B respectively) by automated guided vehicle (AGV) and demonstrate the impact of these two processes to the total processing time in one of the workstation in shop floor system. The simulation system in this study estimates most of the production factors, such as cycle time, production yield, and total number of items in a work order and so on, with the empirical data so as to increase the reliability of the simulated result. The result shows that method A is no worse than method B under most of the production parameters setting. In addition, method A will outperform method B if the number of items in a worker order is more than 1500 or the production yield is lower than 97.5% or number of machines in working is more than 10. And finally, the result indicates the recommended number of machines should be operated is 10, which is a valuable information for producing product and reduce cost.

Completion time: Jan. 2020

IPO model

01. Input

  • Manufacturing data such as processing time for different product id in different stage (CSV file)

02. Process

  • Estimate the processing time of end product and defective, and check the proper distribution for those data
  • Define the detail of the two different feed-material processes done by AGV
  • Use the object-orientated programming to implement the simulation process to mimic the production line in the real world
  • Plot the simulated result

03. Output

  • Simulated result of different feed-material process
  • Graphical and statistical result to show the proper situation for different feed-material processes

Benefit and Result

  • In the simulation system, the simulated result can be generated to estimate the possible outcome in the real manufacturing system.
  • Avoid using the real world system to generate data and save costs
  • Realize the proper situation for different feed-material process so as to tune the manufacturing process.
