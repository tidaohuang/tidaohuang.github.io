Implementation of Simplex Algorithm

Linear Programming

Introduction

After taking the course: Operation Research (I) and linear programming, I put an additional effort to build the Simplex algorithm with python. Users can input the decision variables, objective function (vector c), and constraints (matrix A and right hand side vector b). Then, the program will automatically solve the problem. However, the limitation of this program is that it can only solve the linear programming problem.

Completion time: Nov. 2019

IPO model

01. Input

  • decision variables
  • objective function
  • constraints (right hand side vector b, and matrix A)

02. Process

  • Transform the input data as proper form (in dictionary format of Simplex method)
  • Execute Simplex method to solve the problem
  • [Optional] conduct sensitivity analysis
  • [Optional] re-optimize the problem if any parameter is changed

03. Output

  • The solved solution (either optimal, infeasible, or infinite solution)and the value of its decision variables

Benefit and Result

  • The problem for optimization in the real world can be easily solved if the dimension of the problem is not extremely large and the problem is the linear programming not integer programming or mixed integer programming.
